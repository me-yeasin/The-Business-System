# Analyzing Skill

## Overview
This skill is used for analyzing web traffic potential, optimizing content for search engines (SEO), and evaluating competitor strategies to find content gaps.

## Core Capabilities
- **SEO Optimization:** Structuring HTML and Markdown to prioritize target keywords. Writing optimized meta titles and descriptions.
- **Competitor Content Review:** Analyzing a competitor's blog post or landing page to identify their angle and suggest a superior alternative.
- **Metric Evaluation:** Reviewing campaign data (if provided) to suggest improvements for CTR (Click-Through Rate) and Conversion Rate.

## SEO Best Practices
- **Primary Keywords:** Must be present in the H1, first 100 words, and naturally distributed (density ~1-2%).
- **Semantic HTML:** Ensure proper use of H2/H3 tags for structuring sub-topics.
- **User Intent:** Always map the content to the searcher's intent (Informational, Navigational, Transactional, Commercial Investigation).

## Execution Workflow
1. **Data Intake:** Read the provided content, URL, or data set using web search or file reading tools.
2. **Audit:** Identify weaknesses (e.g., missing keywords, weak hooks, poor readability, missing alt text).
3. **Actionable Recommendations:** Provide a prioritized list of changes. Use exact 'Before & After' examples for clarity.
4. **Implementation:** If requested, directly edit the files or content to apply the SEO and structural improvements.
